# 商标查询H5

#### 运行环境
*  IIS/Apache/Nginx
*  PHP >=5.3
*  MySQL>=5.0

#### 如何运行?
* 修改api.php第6行$appCode值,即可部署使用.

#### appCode如何获取?
1. 百度云市场: [点击前往](https://apis.baidu.com/store/detail/6fb987dd-93eb-41ab-8d18-c54e039c0c7e) 购买
2. 进入 买家中心 [点击前往](https://console.bce.baidu.com/mkt/#/mkt/purchased/list)
3. 点击 对应订单号
4. 点击 交付信息
5. 点击 显示密钥
6. 短信验证后即可显示完整appCode
